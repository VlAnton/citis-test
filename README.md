# Тестовое для СИТИС

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## dev

Запустить сервер разработки на `http://localhost:3000`:

```bash
npm run dev
```

## prod

Для билда

```bash
npm run build
```

Просмотреть билд локально

```bash
npm run preview
```
